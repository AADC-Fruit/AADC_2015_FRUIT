Detailed Description

class of filter for object detection

Der Filter Objekterkennung ermittelt auf Basis der Tiefeninformationen des "Laserscanners" der Xtion die Objekte. Mit dem Property Sensibility l‰sst sich die Schwelle einstellen, ab dem zwei benachbarte Punkte keine Ebene sondern einen Sprung darstellen. Die Ausgabe folgt im tObjData Struct. Dieses stellt ein L-Shape dar. Mit einem Rechten und Linken Punkt. Der Mittlere Punkt ist der Punkt zwischen den beiden Auﬂenpunkten mit der geringsten Entfernung zur Kamera. Mit dem GCL-Offset wird der Versatz der beiden Kamera Systeme kompensiert.
